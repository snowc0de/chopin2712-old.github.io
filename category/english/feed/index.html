That is not a good experience of course but I will explain you how to install all deb, AppImages, sh, Flatpak and some exes in only one command.

I called it AIP it’s Open Source (of course) and you can install it with the following commands:

git clone https://github.com/chopin2712/Auto-Installer-process
cd Auto-Installer-process
sudo sh setup.sh

Once you installed it you can go in the directory who contain the installer and open it in terminal.

Then run the following command:

sudo aip

Then your file will be installed and then removed (except for the .jar and .AppImage files formats who are exceptions because if you remove them you remove the software…)

Don’t have installer(s)?

If you know the package name but you don’t have the installer, there is another thing you can do with AIP.

Just run the following command:

sudo aipn

Then type the package name

Once you see in the terminal the app is installed, stop the script by pressing CTRL + D to prevent the conflicts.

Then your app will be installed if your package name is valid.

Want to uninstall?

There is no functions to uninstall the apps, but you can do it using Synaptic by running the following command:

sudo synaptic

Then you can search your package, check the box and choose complete removal.

Click on Apply and then Confirm to uninstall the package(s).

Want to contribute?

You can get the source code on GitHub on the following link:

https://github.com/chopin2712/Auto-Installer-Process

Don’t works properly?

You can post an issue on the GitHub project to get support or make a feature request. Thanks for using this app.

]]> Create a Minecraft Server never been so easy on Linux http://localhost/2019/07/08/create-a-minecraft-server-never-been-so-easy-on-linux/ Mon, 08 Jul 2019 15:49:10 +0000 http://localhost/?p=22 Hi! This is my real first post on this website. I will explain you how to use my latest creation. It is called MineCraft Server Setup (MCSS) and is an MC simple server installer. Only linux (for the moment)

It’s made with Python and it’s Open Source. You can browse the source on this link. I am very proud of this! It supports Vanilla, CraftBukkit and Spigot at the moment. This is a little tutorial to explain you how to create your very own MC server.

First run the following commands:

git clone https://github.com/chopin2712/MCSS
cd MCSS
chmod +x install.sh
sudo sh install.sh

Then create a new directory on your computer and open it in terminal…

The real magic happen here, you have just to run:

sudo MCSS

First answer 1 for Vanilla or 2 for craftbukkit/spigot.

Second, download the server.jar file on https://mcversions.net , put it in the directory then press enter.

Third, if you want to change the basics settings of Minecraft answer “Y” and modify your settings (TIPS: For online mode change into false to accept cracked versions)

For the “world wide step”, I recommend you to answer “N” to skip this part for the moment because this feature is not stable enough to be used successfully…

Finally your server will start, just wait 20 seconds and you can follow the above IP addresses to join your world.

You can write “stop” to shut down your Minecraft Server and you can run “op USERNAME” to be admin on your server and run all the admins commands on it directly on Minecraft.

It is very simple to use in deed and my future ideas for this app are:

  • Making the world wide feature works perfectly
  • Making available cross-platform
  • Possibly trying to make a visual GUI to use the app (far in the future)
  • Etc, etc

You can contribute and access to this project by clicking on the following link:

https://github.com/chopin2712/MCSS

Already installed? You can update your version to a higher one by running the following script:

git clone https://github.com/chopin2712/MCSS
cd MCSS
sudo sh install.sh
cd ..
sudo rm -r MCSS

NB: If you made a server in a older version, the update don’t have any effect and your server will works properly…

Thanks for your attention and hope you will try this app soon.
