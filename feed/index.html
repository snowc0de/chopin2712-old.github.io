This is my real first post on this website. I will explain you how to use my latest creation. It is called MineCraft Server Setup (MCSS) and is an only linux (for the moment) MC simple server installer.

It’s made with Python and it is Open Source. You can browse the source on this link. I am very proud of it! It supports Vanilla, CraftBukkit and Spigot at the moment. This is a little tutorial to explain you how to create your very own MC server.

First run the following commands:

git clone https://github.com/chopin2712/MCSS
cd MCSS
chmod +x install.sh
sudo sh install.sh

Then create a new directory on your computer and open it in terminal…

The real magic happen here, you have just to run:

sudo MCSS

First answer 1 for Vanilla or 2 for craftbukkit/spigot.

Second, download the server.jar file on https://mcversions.net , put it in the directory then press enter.

Third, if you want to change the basics settings of Minecraft answer “Y” and modify your settings (TIPS: For online mode change into false to accept cracked versions)

For the “world wide step”, I recommend you to answer “N” for the moment because this feature is not stable enough to be used successfully…

Finally your server will begin, just wait 20 seconds and you can follow the above IP addresses to join your world.

You can stop your server by typing “stop” and you can run “op USERNAME” to be admin on your server and run all the admins commands on it directly on Minecraft.

It is very simple to use in deed and my future ideas for this app are:

  • Making the world wide feature works perfectly
  • Making available cross-platform
  • Possibly trying to make a visual GUI to use the app (far in the future)
  • Etc, etc

You can contribute and access to this project by clicking on the following link:

https://github.com/chopin2712/MCSS

Already installed? You can update your version to a higher one by running the following script:

git clone https://github.com/chopin2712/MCSS
cd MCSS
sudo sh install.sh
cd ..
sudo rm -r MCSS

NB: If you made a server in a older version, the update don’t have any effect and your server will works properly…

Thanks for your attention and hope you will try this app soon.
